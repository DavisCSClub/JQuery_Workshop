Using CSS Selectors with JQuery


$('h1') // selects all h1 elements
$('.class-name') // selects all elements with a class of class-name
$('#demo') // selects all elements with an id of demo

Attribute based Selectors


// selects all **h1** elements with an attribute matching the specified value
$('h1[demo-attribute="demo-value"]')
$('[class^="col"]') // Find all elements that start with a string
$('[class*="md"]') // Find all elements where the value contains a string

Positional Selectors


// Selects all a elements that are direct descendants nav element
$('nav > a')
// Selects all a elements that are descendants nav element
// The elements can appear anywhere inside of the element listed first
$('nav a')

Navigating the DOM Tree - Children


// search through, all **direct** descendents of the currently selected item
currentElement.children('a[href^="http://"]');
// All descendants
currentElement.find('a[href^="http://"]')

Navigating the DOM Tree - Siblings


// search through, all siblings after current element
$('#provide-address').nextAll('input').attr('disabled', 'true');
// search through, all siblings before current element
$('#provide-address').prevAll('input').attr('disabled', 'true');

Chaining Event Handlers


// sample JavaScript
$('#phone').focus(function() {
    // Control has focus. Display help
    $('#phone-help').text('Please enter your phone number as all digits');
}).blur(function() {
    // Control lost focus. Clear help
    $('#phone-help').text('');
});
